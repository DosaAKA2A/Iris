( 06 ) • FAQ

Preguntas frecuentes, respuestas directas.

¿Cuánto tarda un proyecto?+

Depende del plan: una landing del plan Esencial se entrega en 1–2 semanas, una web corporativa completa en 3–5 semanas, y los proyectos de e-commerce o plataforma se estiman a medida. El plazo cuenta desde que recibimos contenidos y feedback con fluidez.

¿Cómo funciona el pago?+

Con un 50% inicial para arrancar el proyecto y el 50% restante a la entrega, antes de la publicación. El pago se realiza de forma segura con Stripe y cada proyecto incluye su factura. Los precios mostrados no incluyen IVA.

¿Usáis plantillas?+

No. Cada web se diseña y se programa a medida: dirección de arte, sistema visual y código propios para tu marca. Es lo que permite que el resultado no se parezca a ninguna otra web.

¿Qué necesito tener preparado antes de empezar?+

Lo esencial: una idea clara de qué quieres conseguir, tus textos e imágenes (o disposición a trabajarlos juntos) y tu logotipo si lo tienes. Si te falta algo de esto, lo resolvemos en la fase de descubrimiento.

¿Qué pasa después del lanzamiento?+

Todos los planes incluyen un periodo de soporte post-lanzamiento (30 o 60 días según el plan) para ajustes y correcciones. Después puedes contratar mantenimiento continuo si quieres que sigamos evolucionando la web contigo.

¿El hosting y el dominio están incluidos?+

Se facturan aparte (aproximadamente 100–300 €/año según el proyecto), pero nos encargamos de toda la configuración técnica: dominio, hosting, correo y puesta en producción.
